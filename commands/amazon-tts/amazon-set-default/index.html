    /amazon_set_default¶

    📖 Description¶

    Sets the settings to be used by the /say and /amazon_say commands by default. Default settings are used for people who have not set their own settings for the Amazon Provider (TTS Tool).

    If you're seeking to change your own settings for this provider, you should use /amazon_set_my.

    ❓ Can Be Used By¶

    Can only be used by users with the MANAGE_GUILD permission.

    🌎 Subcommand: language¶

    🔨 Parameters¶

    Running this command requires the following parameters:

    • <value> - Required: The language code to use as the default Amazon Provider (TTS Tool) language.

    🎈 Usage¶

    You can run this command by typing:

    /amazon_set_default language <value>

    For example:

    🗣 Subcommand: voice¶

    🔨 Parameters¶

    Running this command requires the following parameters:

    • <value> - Required: The name of the voice to be used as the default Amazon Provider (TTS Tool) voice.

    🎈 Usage¶

    You can run this command by typing:

    /amazon_set_default voice <value>

    For example:

    🔊 Subcommand: volume¶

    🔨 Parameters¶

    Running this command requires the following parameters:

    • <value> - Required: The volume to use as the default Amazon Provider (TTS Tool) volume. You can choose any of the suggested values when running the command.

    🎈 Usage¶

    You can run this command by typing:

    /amazon_set_default volume <value>

    For example:

    For example:

    ℹ️ Other Information¶

    Some extra information to take into account:

    • Can only be run from a server.
    • Can only be run from a NSFW channel.
    • Saves data from the user.
      • Saves the default settings for the Amazon Provider (TTS Tool) for the server.
